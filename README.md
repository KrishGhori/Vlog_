# VlogSite – Bootstrap Web Project

##  Project Overview

**VlogSite** is a simple multi-page vlog-style website built using **HTML, CSS, and Bootstrap**.
The project demonstrates responsive design, consistent UI layout, and basic multimedia integration such as videos, audio, and image galleries.

This project was created as a **college web development assignment**.

---

##  Features

* Responsive layout using Bootstrap
* Premium gradient navigation bar
* Multi-page website structure
* Video and audio embedding
* Image gallery
* Blog page layout
* Data table for vlog records
* Consistent footer across all pages
* Clean and reusable CSS styling

---

##  Technologies Used

* HTML5
* CSS3
* Bootstrap 5
* Bootstrap Icons

---

##  Project Structure

```
VlogSite/
│
├── index.html        # Home page
├── videos.html       # Video & audio content
├── gallery.html      # Image gallery
├── table.html        # Data table page
├── content.html      # Featured content
├── blog.html         # Blog posts
├── about.html        # About page
└── style.css         # Global styles
```

---

##  Pages Description

### Home

Landing page introducing the vlog website and main categories.

### Videos

Displays vlog videos and audio content.

### Gallery

Shows a collection of images in a responsive grid layout.

### Table

Displays vlog data such as title, category, and views using a Bootstrap table.

### Content

Highlights featured topics like travel, food, and lifestyle.

### Blog

Simple blog layout with article preview.

### About

Information about the creator of the vlog site.
